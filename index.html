Shortest Path Finder AI Project

The finder includes an implementation of an AIModule interface that computes a path from the start location to the end location while minimizing the total search space. The terrain map is a 2-D world layered on top of a rectangular grid. Each point in the world has a height (0 - 255). One can move to any of the eight squares adjacent to the current location. The cost to traverse between tiles depends on the differences in height between the tiles. The finder could either use Dijkstra's algorithm or Astar algorithm to find the optimal path.
