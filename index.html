AtHum Internship:Interactive SVG Images

AtHum is the AI interior designer for real estate. AtHum is adding a new feature to their remote office application. The feature adds interactive actions to the images of properties and floorplans within the project that AtHum deploys to the customers. We were given the responsiblity for deploying this feature in production. The interactive actions include highlighting of selected building section in the image, the menu actions and displaying of callout which includes the information of the selected section and opens particular section page of the property when clicked. To start with, we are given a .svg file which include the interactive image generated by Adobe Illustrator. we needed to deliver .json file or JavaScript object notation file which includes all the actions. The file includes three sections: svgActions which are the actions performed when hovered over the image, second is menu in which the menu and submenu options are includes along with the actions that will be performed if one of the options is selected. The last one is the callout section, which is the detail box on the top left corner, displayed when clicked on the image or second level menu options.
